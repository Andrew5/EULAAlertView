# EULAAlertView
swift版用户协议弹窗
在程序开始，进入主页之前弹出的用户协议、用户须知、隐私政策等弹窗；该事例文字颜色、点击事件、文字大小等都可以更自定义，且不会出现长按弹窗等问题，高度自定义View实现。
